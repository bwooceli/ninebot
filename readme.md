# ninebot

### Quickstart

1) Clone Repo
2) python -m venvs/main_env
3) pip install -r requirements.txt
4) ./venvs/main_env/bin/activate or .\venvs\main_env\scripts\Activate
5) mkdir secret
6) add the secret settings (see settings.py) to secret\base.py
